Recreation of the 1982 arcade Pengo

Game Description

Pengo is a tribute to the classic arcade game originally developed by Sega in 1982. In the game, the player takes control of a penguin trapped in a maze made of ice blocks. The main objective is to push these blocks to crush the enemies known as "Sno-Bees" while avoiding being caught by them. This version has been recreated using C++ and the Raylib graphics library. It includes core mechanics from the original game along with modern features such as visual debugging tools (colliders, active zones, etc.) to support development and testing. The project was developed as part of an academic assignment at UPC and aims to combine retro gameplay with improved design and functionality.
